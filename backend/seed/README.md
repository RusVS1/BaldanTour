Данные инициализации для бэкенда.

- `sample_parsed_tours.json`: небольшая Django-фикстура, которая используется, если CSV не примонтированы.
  - Содержит все строки `ParsedCountry` и небольшой детерминированный набор строк `ParsedAvailableTour`
    (включая `description` и `embedding`).

Полный датасет намеренно не хранится в git (слишком большой). Для полного импорта используйте монтирование CSV
и расчет эмбеддингов на GPU через `RUN_GPU_EMBEDDINGS=true`.
